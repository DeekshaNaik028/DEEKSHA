## ğŸ“ Customization

### Update Personal Information

Edit the data files in the `src/data/` directory:

- **aboutData.js** - Bio and statistics
- **skillsData.js** - Technical skills
- **projectsData.js** - Project information
- **achievementsData.js** - Awards and certifications
- **contactData.js** - Contact information and social links

### Modify Styling

- Update Tailwind configuration in `tailwind.config.js`
- Customize component styles directly in JSX using Tailwind classes
- Add custom CSS in `App.css` or `index.css`
